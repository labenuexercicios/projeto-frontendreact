## Technologies
👩🏻‍💻 Project is created with:
* JSX
* React
* Styled Componentes
* Props
* React Hooks (UseState, UseEffect)
* Data flux
* Lists rendering
* localStorage


## Overview

The Project has 3 great sections:

1. Home
    - Showing a gallery with an array of product cards
    - Showing card with properties of products
    - Function to add card to cart

2. Cart
    - Cart with array of products added
    - Function to remove card from cart
    - Total price of all products inside the cart

3. Filters
    - Per min and max price
    - Per name
    - Price Sorting

## Run & Deploy

These are prerequisites for installing:
-   Node.js
-   npm

To install dependencies in your project directory and run the application in your computer, run the following commands:

```sh
npm install
```

---

```sh
npm run start
```
